# 🐍 React Snake Game (Multiplayer Edition)

Сучасна реалізація класичної гри "Змійка", побудована на **React** та **Vite**. Проєкт створено в рамках навчального завдання з додаванням унікальних механік.

## 🚀 Особливості (Features)

### Базовий функціонал:

- **Grid Engine:** Поле з динамічним рендерингом.
- **Snake Logic:** Класичне переміщення, ріст при поїданні та програш при зіткненні зі стінами або собою.
- **Score System:** Відстеження поточної кількості балів.

### "Отсебятина" (Advanced Features) ⚡:

- **Dual Control:** Керування двома руками (або для двох гравців на одній клавіатурі).
- **Dangerous Mines:** Об'єкти-міни, які зменшують ваш рахунок або вкорочують змійку.
- **Smart Food:** Їжа, яка може "втікати" від гравця, якщо він знаходиться занадто близько.
- **Static Barriers:** Непрохідні перешкоди на полі, що генеруються випадковим чином.

## 🛠 Технології

- [React.js](https://reactjs.org/) — інтерфейс та стан гри.
- [Vite](https://vitejs.dev/) — швидка збірка та HMR.
- [CSS Modules / Tailwind](https://tailwindcss.com/) — для стилізації ігрового поля.
